# Emozzk Lite

CHZZK 채팅 이모티콘 패널을 더 빠르게 조작하기 위한 브라우저 확장 프로그램입니다.

Emozzk Lite는 치지직(CHZZK)의 이모티콘 창을 직접 조작하는 방식으로 동작합니다.
채팅 메시지 payload나 WebSocket 전송 데이터를 수정하지 않고, 사용자가 실제로 클릭할 수 있는 이모티콘 버튼을 단축키와 즐겨찾기 UI로 더 편하게 사용할 수 있도록 돕습니다.

---

## 주요 기능

### 이모티콘 패널 단축키

* `E` 키로 이모티콘 패널을 열고 닫을 수 있습니다.
* 채팅 입력 중에는 단축키가 동작하지 않도록 보호합니다.
* 브라우저 기본 입력, 조합 입력, modifier 키 입력과 충돌하지 않도록 처리합니다.

### 이모티콘 선택 단축키

* 이모티콘 패널이 열린 상태에서 `F1` ~ `F10` 키로 화면에 보이는 이모티콘을 선택할 수 있습니다.
* 이모티콘 버튼을 직접 클릭하는 방식으로 동작합니다.
* 채팅 입력창에 이미 이모티콘이 가득 찬 경우 불필요한 클릭을 줄이도록 처리합니다.

### 즐겨찾기

* 최근 사용한 이모티콘을 즐겨찾기에 등록할 수 있습니다.
* 즐겨찾기 이모티콘은 최근 사용한 이모티콘 섹션 위에 별도 섹션으로 표시됩니다.
* 즐겨찾기 목록내에서는 직접 드래그하여 순서를 바꿀 수 있습니다.
* 즐겨찾기에서 제거해도 CHZZK 기본 최근 이모티콘 목록은 그대로 유지됩니다.

### 단축키 바인딩

* 즐겨찾기 이모티콘에 원하는 단축키를 등록할 수 있습니다.
* 단축키 등록 모드에서 이모티콘을 선택한 뒤 키를 지정하고 저장할 수 있습니다.
* 등록된 단축키는 이모티콘 위에 배지로 표시됩니다.
* 단축키 해제 모드에서 여러 이모티콘을 선택한 뒤 한 번에 해제할 수 있습니다.

### 단축키 세트

* 단축키 바인딩 세트를 2개까지 사용할 수 있습니다.
* 즐겨찾기 목록은 공유하고, 단축키 등록 정보만 세트별로 분리됩니다.
* 즐겨찾기 헤더의 `1`, `2` 세그먼트 버튼으로 활성 세트를 전환할 수 있습니다.
* 세트 전환 시 현재 활성 세트의 단축키와 배지만 표시됩니다.

### 실험실 옵션

* KeyUp 단축키를 허용할 수 있습니다.
* KeyDown + KeyUp 동시 지정 기능을 사용할 수 있습니다.
* 해당 옵션은 팝업 설정에서 켜고 끌 수 있습니다. 확장기능 -> Emozzk Lite

### 최근 이모티콘 보존 개수 확장

* CHZZK의 최근 이모티콘 목록을 더 많이 유지할 수 있도록 보존 개수를 조절할 수 있습니다.
* 팝업 설정에서 최근 이모티콘 보존 개수를 설정할 수 있습니다.
* 변경된 보존 개수는 치지직 페이지를 새로고침한 뒤 적용됩니다.

---

## 동작 방식

Emozzk Lite는 다음 원칙을 기준으로 동작합니다.

* CHZZK 채팅 WebSocket payload를 직접 수정하지 않습니다.
* 구독 권한을 우회하지 않습니다.
* 이모티콘 입력은 CHZZK 이모티콘 패널의 실제 버튼을 클릭하는 방식으로 처리합니다.
* 확장 프로그램의 즐겨찾기, 단축키, 단축키 세트 정보는 브라우저 저장소에 저장됩니다.
* CHZZK의 최근 이모티콘 목록은 필요한 범위에서만 읽거나 보존 개수 확장을 위해 처리합니다.

| 기능            | 키            |
| ------------- | ------------ |
| 이모티콘 패널 열기/닫기 | `E`          |
| 즐겨찾기 단축키      | 사용자가 직접 지정   |

채팅 입력창에 커서가 있거나 텍스트 입력 중인 경우, 단축키가 의도치 않게 동작하지 않도록 제한됩니다.


주요 파일 역할:

| 파일                                 | 역할                          |
| ---------------------------------- | --------------------------- |
| `emote-favorites-render.js`        | 즐겨찾기 섹션 렌더링, 단축키 세트 전환 UI   |
| `emote-favorites-events.js`        | 즐겨찾기 등록/삭제 이벤트              |

| `shortcut-storage.js`              | 단축키 바인딩과 단축키 세트 저장          |
| `shortcut-controller.js`           | 키 입력 감지 및 단축키 실행            |
| `badge-overlay.js`                 | 이모티콘 위 단축키/해제 배지 표시         |
| `recent-emote-storage.js`          | CHZZK 최근 이모티콘 storage 읽기/쓰기 |
| `favorite-recent-emote-storage.js` | Emozzk Lite 즐겨찾기 저장         |
| `extension-settings-storage.js`    | 팝업 설정 저장 및 동기화              |
| `inject.js`                        | 페이지 컨텍스트에서 필요한 보조 처리        |

## 저장 데이터

Emozzk Lite는 확장 기능 동작을 위해 브라우저 저장소를 사용합니다.

저장되는 주요 정보:

* 즐겨찾기 이모티콘 목록
* 즐겨찾기 정렬 순서
* 단축키 바인딩
* 활성 단축키 세트
* 실험실 옵션
* 최근 이모티콘 보존 개수 설정

---

## 주의 사항

* 이 확장 프로그램은 CHZZK 공식 기능이 아닙니다.
* CHZZK 웹사이트 구조가 변경되면 일부 기능이 동작하지 않을 수 있습니다.
* 이모티콘 입력은 CHZZK UI 버튼 클릭을 기반으로 하므로, CHZZK 내부 정책이나 입력 제한을 우회하지 않습니다.
* 최근 이모티콘 보존 개수 확장 기능은 CHZZK의 최근 이모티콘 저장 구조에 의존합니다. CHZZK 구조 변경 시 동작이 달라질 수 있습니다.

---

## 비공식 고지

본 확장 프로그램은 NAVER 또는 CHZZK와 관련이 없는 비공식 도구입니다.
NAVER, CHZZK 및 관련 명칭은 각 소유자의 상표입니다.

본 확장 프로그램 사용으로 인해 발생하는 결과에 대한 책임은 사용자에게 있습니다.
