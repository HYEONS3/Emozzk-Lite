# Emozzk Lite 개인정보 처리방침

Emozzk Lite는 CHZZK 기본 이모티콘 패널을 키보드 단축키와 즐겨찾기 기능으로 더 빠르게 조작할 수 있게 해주는 Chromium 계열 브라우저 확장 프로그램입니다.

Emozzk Lite는 「개인정보 보호법」 제30조에 따라 이용자의 개인정보를 보호하고, 개인정보 처리와 관련한 사항을 명확히 안내하기 위하여 다음과 같이 개인정보 처리방침을 수립·공개합니다.

이 확장 프로그램은 CHZZK 또는 NAVER의 공식 도구가 아니며, CHZZK 또는 NAVER와 제휴하거나 보증을 받은 제품이 아닙니다.

---

## 제1조 개인정보의 처리 목적 및 항목

Emozzk Lite는 서비스 제공을 위해 필요한 최소한의 기능 설정만 사용자의 브라우저에 저장합니다.

Emozzk Lite는 서버를 운영하여 개인정보를 수집하거나 저장하지 않으며, 사용자의 개인정보를 외부 서버로 전송하지 않습니다.

| 구분 | 처리 항목 | 처리 목적 | 저장 위치 | 외부 전송 여부 |
| --- | --- | --- | --- | --- |
| 즐겨찾기 | 이모티콘 식별자, 이모티콘 이미지 URL | 즐겨찾기 이모티콘 표시 및 선택 | 사용자 브라우저 로컬 저장소 | 전송하지 않음 |
| 단축키 설정 | 단축키 키 값, 단축키 세트, 세트 이름, 이모티콘 식별자 | 이모티콘 단축키 등록 및 실행 | 사용자 브라우저 로컬 저장소 | 전송하지 않음 |
| 확장 프로그램 설정 | 최근 이모티콘 보존 개수, 단축키 세트 수, 실험실 옵션 | 사용자 설정 유지 | 사용자 브라우저 로컬 저장소 | 전송하지 않음 |
| 최근 이모티콘 보조 정보 | CHZZK 최근 이모티콘 목록 중 기능 동작에 필요한 정보 | 최근 이모티콘 보존 개수 관리 및 즐겨찾기 우선 보존 | 사용자 브라우저 로컬 저장소 | 전송하지 않음 |

Emozzk Lite는 다음 정보를 수집, 저장, 전송하지 않습니다.

- 채팅 메시지 내용
- 로그인 정보
- 비밀번호
- 쿠키
- 결제 정보
- 위치 정보
- 건강 정보
- 연락처
- 브라우징 기록
- 사용자를 직접 식별할 수 있는 개인정보
- CHZZK 메시지 payload
- WebSocket 전송 데이터
- 구독 권한 정보

Emozzk Lite는 사용자가 실제로 클릭할 수 있는 CHZZK 이모티콘 버튼을 확장 프로그램 UI와 단축키로 더 편하게 누르는 방식으로 동작합니다.

## 제2조 개인정보의 처리 및 보유 기간

Emozzk Lite는 개인정보를 외부 서버에 저장하지 않습니다.

즐겨찾기, 단축키, 최근 이모티콘 설정 등 기능 동작에 필요한 설정값은 사용자의 브라우저 로컬 저장소에 저장되며, 사용자가 직접 삭제하거나 확장 프로그램을 제거할 때까지 보관될 수 있습니다.

사용자는 다음 방법으로 저장된 설정을 삭제할 수 있습니다.

1. 브라우저에서 Emozzk Lite 확장 프로그램을 제거
2. 브라우저 확장 프로그램 저장소 데이터 삭제
3. 브라우저 사이트 데이터 또는 로컬 저장소 삭제
4. Emozzk Lite 설정 또는 초기화 기능이 제공되는 경우 해당 기능 사용

확장 프로그램을 제거하면 브라우저 정책에 따라 확장 프로그램이 저장한 로컬 데이터가 함께 삭제될 수 있습니다.

---

## 제3조 개인정보의 제3자 제공

Emozzk Lite는 이용자의 개인정보를 제3자에게 제공하지 않습니다.

또한 Emozzk Lite는 사용자의 즐겨찾기, 단축키, 최근 이모티콘 설정, 채팅 관련 정보를 외부 서버나 제3자에게 전송하지 않습니다.

다만 법령에 따라 제출 의무가 발생하는 경우에는 관련 법령에서 정한 절차에 따를 수 있습니다.

---

## 제4조 개인정보 처리의 위탁 및 국외 이전

Emozzk Lite는 개인정보 처리 업무를 제3자에게 위탁하지 않습니다.

Emozzk Lite는 개인정보를 국외로 이전하지 않습니다.

확장 프로그램의 소스 코드, 배포 페이지, 문서 페이지가 GitHub 또는 Chrome Web Store 등 외부 플랫폼을 통해 제공될 수 있으나, Emozzk Lite가 사용자의 기능 설정이나 개인정보를 해당 플랫폼으로 전송하는 것은 아닙니다.

---

## 제5조 정보주체의 권리 및 행사 방법

사용자는 언제든지 자신의 브라우저에 저장된 Emozzk Lite 설정 데이터를 삭제할 수 있습니다.

Emozzk Lite는 개인정보를 외부 서버에 저장하지 않으므로, 별도의 서버 내 개인정보 열람, 정정, 삭제, 처리정지 절차는 발생하지 않습니다.

다만 사용자는 다음 조치를 통해 저장된 데이터를 직접 관리할 수 있습니다.

- 확장 프로그램 제거
- 브라우저 확장 프로그램 데이터 삭제
- 브라우저 로컬 저장소 삭제
- CHZZK 사이트 데이터 삭제

개인정보 처리와 관련한 문의가 필요한 경우 본 방침의 연락처로 문의할 수 있습니다.

---

## 제6조 개인정보의 안전성 확보 조치

Emozzk Lite는 개인정보를 외부 서버에 수집하거나 저장하지 않습니다.

기능 동작에 필요한 설정값은 사용자의 브라우저 로컬 저장소에 저장되며, 브라우저와 운영체제의 보안 체계에 따라 관리됩니다.

Emozzk Lite는 다음 원칙을 따릅니다.

- 필요한 권한만 사용합니다.
- CHZZK 메시지 전송 데이터나 WebSocket payload를 수정하지 않습니다.
- 사용자의 채팅 메시지, 로그인 정보, 쿠키, 브라우징 기록을 수집하지 않습니다.
- 기능 제공에 필요한 설정값을 외부 서버로 전송하지 않습니다.
- 단축키와 즐겨찾기 기능은 사용자가 실제로 접근 가능한 CHZZK 이모티콘 버튼을 클릭하는 방식으로 동작합니다.

---

## 제7조 쿠키 및 자동 수집 장치의 사용

Emozzk Lite는 자체적으로 쿠키를 생성하거나 사용하지 않습니다.

또한 Emozzk Lite는 이용자를 추적하기 위한 식별자, 분석 도구, 광고 추적 도구를 사용하지 않습니다.

다만 CHZZK 웹사이트 자체가 쿠키, 로컬 저장소, 세션 저장소 등을 사용할 수 있으며, 이는 CHZZK 또는 NAVER의 정책에 따릅니다.

---

## 제8조 Chrome Web Store 사용자 데이터 정책 관련 고지

Emozzk Lite는 Chrome Web Store 사용자 데이터 정책의 제한적 사용 원칙에 따라, 사용자 데이터가 필요한 경우에도 공개된 목적 범위 내에서만 사용합니다.

Emozzk Lite는 다음 행위를 하지 않습니다.

- 사용자 데이터를 판매하지 않습니다.
- 사용자 데이터를 광고 목적으로 사용하지 않습니다.
- 사용자 데이터를 제3자에게 전송하지 않습니다.
- 사용자 데이터를 신용평가, 대출, 보험, 고용, 주거 등과 관련한 판단에 사용하지 않습니다.
- 사용자의 채팅 내용, 로그인 정보, 쿠키, 결제 정보, 위치 정보, 브라우징 기록을 수집하지 않습니다.

Emozzk Lite가 사용하는 데이터는 확장 프로그램 기능 제공을 위한 로컬 설정값에 한정됩니다.

## 제9조 확장 프로그램 권한

Emozzk Lite는 기능 제공을 위해 다음 권한을 사용할 수 있습니다.

| 권한 | 사용 목적 |
| --- | --- |
| `storage` | 즐겨찾기, 단축키, 확장 프로그램 설정을 사용자 브라우저에 저장 |
| `https://chzzk.naver.com/*` | CHZZK 페이지에서 이모티콘 패널 조작, 즐겨찾기 UI 표시, 단축키 기능 실행 |

Emozzk Lite는 위 권한을 사용하더라도 사용자의 개인정보를 외부 서버로 전송하지 않습니다.
